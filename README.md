# Jobs-RAG-AI-Agent
A Custom Python-built RAG AI Agent for jobs

### To start:
You will need docker v2 

-   docker compose up


### To run the Job Ingestor:
-   docker compose exec job_ingestor python job_ingestor/ingest.py

###Go to the UI
localhost:3000
